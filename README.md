# Утилита фильтрации содержимого файлов

Консольная утилита читает несколько входных файлов, содержащих вперемешку **целые числа**, **вещественные числа** и **строки** (разделитель — перевод строки), и записывает данные разных типов в разные выходные файлы.

Строки из файлов читаются по очереди в соответствии с порядком перечисления файлов в командной строке.

## Функциональность

- **integers** → в файл `integers.txt`
- **floats** → в файл `floats.txt`
- **strings** → в файл `strings.txt`

По умолчанию выходные файлы создаются в текущей директории.  
Файлы результатов создаются **только по мере необходимости**: если какого-то типа данных нет — пустой выходной файл не создаётся.

Поддерживаются опции:
- `-o <path>` — директория для выходных файлов
- `-p <prefix>` — префикс имён выходных файлов
- `-a` — добавление в существующие выходные файлы (иначе перезапись)
- `-s` — краткая статистика
- `-f` — полная статистика

Статистика выводится в консоль для каждого типа данных.

## Требования

- **Java:** 21
- **Система сборки:** **Apache Maven 3.9+**
- **Сторонние библиотеки:** не используются (только стандартная библиотека Java)

## Сборка

```bash
mvn clean package
```


## Пример запуска утилиты

```bash
java -jar target/filter-1.0-SNAPSHOT.jar -s -a -p sample- in1.txt in2.txt
```

## Обработка ошибок

- Если входной файл не найден / недоступен / не является обычным файлом — выводится сообщение об ошибке, и обработка продолжается для остальных файлов.
- Если невозможно создать/использовать директорию вывода (`-o`) или невозможно записывать выходные файлы — программа сообщает причину и завершает работу с ненулевым кодом.

## Примечания по определению типов

- **Целое число (integers)** — строка формата `+/-` и только цифры (например: `45`, `-10`, `+7`). Размер числа не ограничен (тип определяется по формату строки, без `long`).
- **Вещественное число (floats)** — строка, которая успешно парсится как `BigDecimal` и не является целым по формату. Поддерживается научная запись (например: `1.528535047E-25`).
- **Всё остальное** считается строкой (**strings**) и записывается как есть.
